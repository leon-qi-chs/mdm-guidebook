## 项目概述

### 背景
最近产品表经常超时，导致影响了 Golden Pipe 的正常流程。

### 目标
简化产品 ETL，将产品 ETL 的计算下放到 Databricks，提升处理性能，避免超时问题。

## 技术方案

### 优化策略
1. **计算下放**：将复杂计算从上游移至 Databricks
2. **并行处理**：利用 Databricks 分布式计算能力
3. **查询优化**：优化 SQL 查询，减少数据扫描量
4. **增量处理**：减少全量计算频率

### 架构调整
```
原架构: 
上游系统 → 复杂 ETL → Golden Pipe

优化后:
上游系统 → 简化 ETL → Databricks 计算 → Golden Pipe
```
